NHL Players' Birth Cities

Introduction

While hockey is stereotypically a sport played by those from the True North , in practice players in the NHL come from all over the northern hemisphere.

This map displays the birth city of every NHL player to have played in at least one game of the 2017-18 season, as of 2017-12-06.

Instructions

The icons to the left allow you to search and filter the map by player, team, or city, respectively. More instructions can be found in each individual tab.

Methodology

Birth city data was taken from the excellent Hockey Reference site, and the city name was geocoded against the Geonames API to get the city's coordinates. I performed a little cleanup on the data by hand, as there were a few weird outliers that were very obviously wrong. The Geonames API also has a tendency to choose a municipal airport in some cases, which led to some other weirdness; for example, Sidney Crosby's birth town of Cole Harbor was geocoded to the local airport. I didn't make as much of an effort to clean these up, as by and large they're rather close to where they should be. You'll surely find some other things if you explore the data closely enough. You can find the code I used to perform the data processing on Github.
