NHL Players' Birth Cities

This map displays the birth city of (nearly) every NHL player to have played in at least one game of the 2017-18 season, as of 2017-12-06.

The icons to the left allow you sort search & filter the map by player, team, or city, respectively.

Birth city data was taken from the excellent Hockey Reference site, and the city name was geocoded against the Geonames API to get the city's coordinates. I performed very little cleanup on the data, which resulted in some of the outliers that you see. The Geonames API also has a tendency to choose a municipal airport in some cases, which led to some other weirdness. You'll surely find some other things if you explore the data closely enough. You can find the code I used to perform the data processing on Github.

Information & Credits

First and foremost, thanks to Leaflet for creating an amazing free & open source mapping library.

This page uses Leaflet-sidebar, Leaflet-ajax, and OverlappingMarkerSpiderifier-Leaflet to work. Many thanks to the creators of those libraries, doing a bunch of work that I didn't have to.

I had never heard of or used the Geonames API before this project, and without their free city geocoding API, this project would have been DOA. I highly recommend using & contributing to their service.

Finally, thanks to Hockey-Reference for maintaining one of the most comprehensive NHL stats sites on the web, and making that data freely available to all.
